.tabWrap,
.tabNav,
.tabContent		 스크립트 내부용
(변경하거나 없으면 기능작동 안함)
.tab-base 기본 스타일 - 타입1 (스타일이 다를 경우 삭제 후 자유롭게 네이밍)
.tab-base2 기본 스타일 - 타입2 (스타일이 다를 경우 삭제 후 자유롭게 네이밍)
.tab-feature 실행/초기화로 사용할 선택자(자유롭게 네이밍). 하나의 문서에서 중복 X
탭과 내용의 매칭(접근성) a[id] == .tabContent[aria-labelledby] 
<div class="tabWrap tab-base tab-feature">
	<ul class="tabNav">
		<li><a href="#" id="tabNav1" role="tab">지배구조</a></li>
		<li><a href="#" id="tabNav2" role="tab">기금규모</a></li>
		<li><a href="#" id="tabNav3" role="tab">기금수익률</a></li>
		<li><a href="#" id="tabNav4" role="tab">자산배분</a></li>
	</ul>
	<div class="tabContent" role="tabpanel" aria-labelledby="tabNav1">
		지배구조 내용
	</div>
	<div class="tabContent" role="tabpanel" aria-labelledby="tabNav2">
		기금규모 내용
	</div>
	<div class="tabContent" role="tabpanel" aria-labelledby="tabNav3">
		기금수익률 내용
	</div>
	<div class="tabContent" role="tabpanel" aria-labelledby="tabNav4">
		자산배분 내용
	</div>
</div>
<script>
commonUI.tabNav('.tab-feature', 1);
</script>
