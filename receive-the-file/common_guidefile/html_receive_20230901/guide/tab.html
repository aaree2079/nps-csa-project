.tab-base 기본 스타일 적용 (스타일이 다를 경우 삭제 후 자유롭게 네이밍)
.tab-feature 실행/초기화로 사용할 선택자(자유롭게 네이밍). 하나의 문서에서 중복 X
탭과 내용의 매칭(접근성) a[id] == .tabContent[aria-labelledby]
