하이픈 & 대시
  • 하이픈-문장 부호의 하나. ‘－’의 이름이다. 차례대로 이어지는 내용을 하나로 묶어 열거할 때 쓰거나, 두 개 이상의 어구가 밀접한 관련이 있음을 나타낼 때 쓴다.
  • 대시-문장 부호의 하나. ‘—’의 이름이다. 제목 다음에 표시하는 부제의 앞뒤에 쓴다.
